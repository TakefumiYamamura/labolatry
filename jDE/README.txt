This package is source code of jDE and dynNP-jDE.
About these methods, please see following papers:

1. J. Brest, S. Greiner, B. Boskovic, M. Mernik, and V. Zumer, “Self-adapting control parameters in differential evolution: A comparative study on numerical benchmark problems,” IEEE Tran. Evol. Comput., vol. 10, no. 6, pp. 646–657, 2006.

2. J. Brest and M. S. Maucec, “Population size reduction for the differential evolution algorithm,” Applied Intelligence, vol. 29, no. 3, pp. 228–247, 2008.

##############################
System configurations in our experimental enviroment:

OS: Ubuntu 12.04 LTS
CPU: core i7 (2.20GHz)
RAM: 8GB
Language: C++
Compiler: g++ (gcc) with -O3 optimization flag

##############################
How to execute:

Step 1. #make
Step 2. #./jde_exe

Dimension size and parameter settings are easily mutable by rewriting source code in de_test.cc.
